## Side Channel Analysis 101 (SCA101) Labs

* **Group A HW**: CW-Lite (XMEGA or ARM), SCAPACK-L1, SCAPACK-L2, CW1200 ($250 - $3800)
* **CW-Nano**: ChipWhisperer-Nano ($50)
* **Simulator**: Pre-recorded power traces (FREE)
